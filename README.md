# FlourApp-TrabajoProfesional
Sistema para Harinas Lolato

# Instalación
1. Instalar node.js (última version que viene con npm)
2. Descargar mongoDB y ejecutar el archivo "mongod"
3. Descargar los archivos del repo (o hacer clone)
4. Ejecutar "npm install" parado sobre el directorio raiz
5. Ejecutar "npm start"
